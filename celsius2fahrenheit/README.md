# 🌡️ Celsius to Fahrenheit Converter

A simple C++ command-line tool that converts temperature from **Celsius** to **Fahrenheit** using the standard conversion formula.

---

## 🧠 Formula Used

\[
°F = (°C × \frac{9}{5}) + 32
\]

---

## 🚀 How to Run

### Requirements:
- A C++ compiler (e.g., `g++` or `clang++`)
